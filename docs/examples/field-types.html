字段类型示例

展示 SDK 当前支持的所有字段类型：string、number、boolean、image、video、file、array，以及对应的 configJson 写法和取值方式。

完整 configJson

以下示例包含了所有支持的字段类型：

import { defineTemplateConfig } from 'auto-exhibition-template-sdk/config'

const configJson = defineTemplateConfig({
  meta: {
    name: '字段类型示例',
    code: 'field-types-demo',
    version: '1.0.0'
  },
  dataSchema: {
    fields: [
      // string 类型
      { key: 'title', type: 'string', label: '标题', value: '默认标题' },

      // number 类型
      { key: 'count', type: 'number', label: '数量', value: 3 },

      // boolean 类型
      { key: 'enabled', type: 'boolean', label: '是否启用', value: true },

      // image 类型
      { key: 'poster', type: 'image', label: '海报', value: '/assets/poster.png' },

      // video 类型
      { key: 'trailer', type: 'video', label: '视频', value: 'https://example.com/video.mp4' },

      // file 类型
      { key: 'guideFile', type: 'file', label: '附件', value: '/assets/guide.pdf', accept: ['audio', '.pdf', '.zip'] },

      // array 类型
      {
        key: 'timeline',
        type: 'array',
        label: '时间轴',
        operations: ['add', 'delete'],
        value: [
          { key: 'phase01', type: 'string', label: '阶段一', value: '接入边缘传感节点。' },
          { key: 'phase02', type: 'string', label: '阶段二', value: '将采集数据转换为状态向量。' }
        ]
      }
    ]
  },
  functions: {}
})

export default configJson

string 字段

普通文本字段，value 为字符串。

// configJson
{ key: 'title', type: 'string', label: '标题', value: '默认标题' }

// 组件取值
const title = useTemplateValue('title', '')
// title 是 string 类型

number 字段

数字字段，value 为数字。

// configJson
{ key: 'count', type: 'number', label: '数量', value: 3 }

// 组件取值
const count = useTemplateValue('count', 0)
// count 是 number 类型

boolean 字段

布尔字段，value 为 true 或 false。

// configJson
{ key: 'enabled', type: 'boolean', label: '是否启用', value: true }

// 组件取值
const enabled = useTemplateValue('enabled', false)
// enabled 是 boolean 类型
<template>
  <section v-if="enabled">
    <!-- 仅在 enabled 为 true 时显示 -->
  </section>
</template>

image 字段

图片字段，configJson 中 value 为字符串路径。

// configJson
{ key: 'poster', type: 'image', label: '海报', value: '/assets/poster.png' }

// 组件取值
const poster = useTemplateValue('poster', { url: '' })

video 字段

视频字段，configJson 中 value 为字符串路径或 URL。

// configJson
{ key: 'trailer', type: 'video', label: '视频', value: 'https://example.com/video.mp4' }

// 组件取值
const trailer = useTemplateValue('trailer', { url: '', poster: '' })

file 字段

文件字段，configJson 中 value 为字符串路径。可通过 accept 约束允许上传的文件类型。

// configJson
{
  key: 'guideFile',
  type: 'file',
  label: '附件',
  value: '/assets/guide.pdf',
  accept: ['audio', '.pdf', '.zip']
}

// 组件取值
const guideFile = useTemplateValue('guideFile', '')
accept 规则： 内置文件类型当前支持 image、video、audio；具体后缀必须带 . 前缀，例如 .pdf、.glb、.zip。不填写 accept 时不限制文件类型。

array 字段

数组字段，value 为字段对象数组，子项结构与普通字段一致。详细用法见数组字段示例。

// configJson
{
  key: 'timeline',
  type: 'array',
  label: '时间轴',
  operations: ['add', 'delete'],
  value: [
    { key: 'phase01', type: 'string', label: '阶段一', value: '接入边缘传感节点。' }
  ]
}

// 组件取值 — 取整个数组
const timeline = useTemplateValue('timeline', [])

// 组件取值 — 取数组中某一项的字段
const phase01 = useTemplateValue('timeline[0].phase01', '')
