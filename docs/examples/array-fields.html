数组字段示例

展示数组字段的定义方式、operations 控制、以及如何在组件中按路径读取数组字段的值。

基础数组字段

数组字段的 value 是字段对象数组，子项结构与普通字段完全一致。

configJson

import { defineTemplateConfig } from 'auto-exhibition-template-sdk/config'

const configJson = defineTemplateConfig({
  meta: { name: '数组示例', code: 'array-demo' },
  dataSchema: {
    fields: [
      {
        key: 'timeline',
        type: 'array',
        label: '时间轴',
        operations: ['add', 'delete'],
        value: [
          { key: 'phase01', type: 'string', label: '阶段一', value: '接入边缘传感节点。' },
          { key: 'phase02', type: 'string', label: '阶段二', value: '将采集数据转换为状态向量。' }
        ]
      }
    ]
  },
  functions: {}
})

export default configJson

取整个数组

<script setup>
import { useTemplateValue } from 'auto-exhibition-template-sdk'

const timeline = useTemplateValue('timeline', [])
</script>

<template>
  <ul>
    <li v-for="item in timeline" :key="item.key">
      {{ item.value }}
    </li>
  </ul>
</template>

按索引取数组中某项字段

使用点路径 + 中括号索引直接读取特定子项的值：

const phase01 = useTemplateValue('timeline[0].phase01', '')
const phase02 = useTemplateValue('timeline[1].phase02', '')

operations 控制

operations 决定平台侧能对数组进行的操作：

写法 效果
不填写 operations 默认允许 add 和 delete
operations: [] 不允许任何操作，数组项固定
operations: ['add'] 只允许添加，不允许删除
operations: ['delete'] 只允许删除，不允许添加
operations: ['add', 'delete'] 允许添加和删除

固定项数的数组

{
  key: 'slides',
  type: 'array',
  label: '幻灯片',
  operations: [],  // 不允许增删，项数固定
  value: [
    { key: 'slide01', type: 'image', label: '第一张', value: '/assets/slide1.png' },
    { key: 'slide02', type: 'image', label: '第二张', value: '/assets/slide2.png' }
  ]
}

嵌套数组

数组字段的子项同样可以是 array 类型，实现多层嵌套。

{
  key: 'chapters',
  type: 'array',
  label: '章节列表',
  value: [
    {
      key: 'chapter01',
      type: 'array',
      label: '第一章',
      value: [
        { key: 'title', type: 'string', label: '章节标题', value: '第一章：起点' }
      ]
    }
  ]
}

// 按路径读取嵌套值
const chapterTitle = useTemplateValue('chapters[0].title', '')
