媒体字段示例

展示 image、video、file 三种媒体类字段的 configJson 写法和组件取值方式。

image 字段

image 字段在 configJson 里只接受字符串路径（相对路径或绝对 URL）。

configJson

import { defineTemplateConfig } from 'auto-exhibition-template-sdk/config'

const configJson = defineTemplateConfig({
  meta: { name: '媒体示例', code: 'media-demo' },
  dataSchema: {
    fields: [
      {
        key: 'poster',
        type: 'image',
        label: '主视觉海报',
        value: '/assets/template/poster-core.svg'
      }
    ]
  },
  functions: {}
})

export default configJson

组件取值

<script setup>
import { useTemplateValue } from 'auto-exhibition-template-sdk'

const poster = useTemplateValue('poster', { url: '' })
</script>

<template>
  <img :src="poster?.url" alt="海报" />
</template>

video 字段

video 字段在 configJson 里只接受字符串路径（相对路径或绝对 URL）。

configJson

{
  key: 'trailer',
  type: 'video',
  label: '宣传视频',
  value: 'https://interactive-examples.mdn.mozilla.net/media/cc0-videos/flower.mp4'
}

组件取值

<script setup>
import { useTemplateValue } from 'auto-exhibition-template-sdk'

const trailer = useTemplateValue('trailer', { url: '', poster: '' })
</script>

<template>
  <video :src="trailer?.url" :poster="trailer?.poster" controls></video>
</template>

file 字段

file 字段在 configJson 里只接受字符串路径。可通过 accept 约束允许上传的文件类型。

configJson

{
  key: 'guideFile',
  type: 'file',
  label: '讲解文档',
  value: '/assets/template/guide.pdf',
  accept: ['image', '.pdf']
}

accept 可用值

写法 说明 示例
内置类型 限制为某类媒体文件 "image"、"video"、"audio"
具体后缀 必须带 . 前缀 ".pdf"、".glb"、".zip"

组件取值

<script setup>
import { useTemplateValue } from 'auto-exhibition-template-sdk'

const guideFile = useTemplateValue('guideFile', '')
</script>

<template>
  <a :href="guideFile" target="_blank">\u4e0b\u8f7d\u6587\u6863</a>
</template>
