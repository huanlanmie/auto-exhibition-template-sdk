API

template-sdk 提供了一套简洁而强大的 API，涵盖了配置定义、构建处理和运行时取值等核心功能。

SDK 入口

SDK 通过以下方式导入： 

// 默认导入（包含 Vue 插件和所有导出）
import TemplateSdk from 'template-sdk'

// 按需导入
import { useTemplateValue, validateTemplateConfig } from 'template-sdk'
