## 发布结果

- 当前默认 registry 为 `https://registry.npmmirror.com`，同时 `https://registry.npmjs.org` 的登录检查也返回未授权。

## 后续处理

具备 npm 发布权限后，可以在本仓库执行：

```bash
npm login --registry https://registry.npmjs.org
npm publish --registry https://registry.npmjs.org --access public
```

也可以直接使用已生成的本地 tarball 进行内网安装或人工发布。
